MLB Records 2017 Season


The purpose of this scatter plot to show Major League Baseball teams record's over the first ten weeks of the 2017 season. If you hover over the data points, the team and their record during that week will be displayed to the right of the data point.

This plot was created by a script that utilizes D3.js. The script parses a .tsv file that contained the team's name, division, and record on any given week; then the plot is generated.
